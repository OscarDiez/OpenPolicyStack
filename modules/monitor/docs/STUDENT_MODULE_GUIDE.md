# OpenPolicyStack — Student Module Guide

Each student module lives under: modules/<module-name>/.

Minimum deliverables per module:
- README: what it does + how to run it
- Clear inputs/outputs (KPIs/plots/brief)
- A small demo scenario/data
- If exposing an API: endpoint sketch + example request/response

Workflow:
1) Create a branch
2) Push branch
3) Open a PR
4) Get review (CODEOWNERS)
5) Merge (maintainers only)
